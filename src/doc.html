API:
    https://l.esthe.win?name=NAME&to=URL[&password=PASSWORD]
        NAME: the name of the short URL
        URL: the URL to redirect to
        PASSWORD: the password to use for the short URL. (optional)

Password protected shortlinks:
    - Recommended!
    - Pass a password=PASSWORD
    - Can override unprotected shortlinks
    - Can only be overridden by passing the same password

Unprotected shortlinks:
    - Can override other unprotected shortlinks

Example:
    https://l.esthe.win?name=home&to=https://esthe.win/
        => https://l.esthe.win/home => https://esthe.win/
